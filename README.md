jpregel-aws
==============

jPregel on AWS

This code is based on a project called jPregel, written by Peter Cappello, 
and a Java package to run jPregel on AWS, which was written by Varsha Parthasarathy. 
The latter package has been essentially rewritten by Charles Munger. 

For more info, a [tutorial](http://charlesmunger.github.com/jpregel-aws/tutorial.html) on use, and the [API documentation](http://charlesmunger.github.com/jpregel-aws/javadoc-api/index.html), check out the [project page](http://charlesmunger.github.com/jpregel-aws/).
